Marcação de Atendimento

  • Layer: Marcação de Atendimento
  • Cor: Variada
  • Espessura: 30

O objetivo da utilização de uma rede FTTH (fiber to the home) é levar soluções completas de voz, transmissão de dados e internet para os clientes através de uma rede composta por cabos ópticos passivos, splitters divisores de sinais e caixas terminais para alojar os splitters. Os splitters ópticos são elementos capazes de dividir o sinal de uma única fibra em mais saídas para que possa ser possível fazer a ramificação da rede. Os tipos mais comuns são os splitters de 1:2, em que de uma única fibra o sinal é dividido para duas, os splitters de 1:4, em que de uma única fibra o sinal é dividido para outras quatro, os splitters de 1:8, de uma única fibra o sinal é dividido para outras oito, os splitters de 1:16 e 1:32, sendo que esses dois últimos são raramente utilizados e os splitters de 1:4 e 1:8 os mais utilizados pelo seu baixo custo e eficiência. Os splitters de 1:4 são mais utilizados para ramificações dos cabos denominando-se como splitters de derivação/distribuição e os splitters de terminação, assim como são chamados os splitters de 1:8 são mais utilizados para o atendimento final dos clientes.

A marcação de atendimento é o processo no qual é definido o que será atendido pela caixa de terminação (CTO). Essa marcação leva em consideração não só o número máximo de clientes que uma caixa CTO com um splitter de 1:8 pode atender, ela leva em consideração um fator chamado penetração. O fator de penetração leva em consideração a forma de abordagem que a empresa pretende atender em determinadas áreas e é definido pela coordenação de projetos. Em áreas consideradas de padrões elevados o fator de penetração será maior por existir potenciais consumidores de internet e em áreas com padrões médios a procura é consideravelmente menor, consequentemente o fator de penetração será menor. Para isso os fatores de penetração variam de 65 a 100% nas áreas consideradas boas, de 50 a 60% para as áreas medianas e 45% nas áreas consideradas ruins. Isso quer dizer que se há uma área com fator de 50% deve-se marcar 16 clientes para uma caixa que atenderia no máximo 8 e se o fator for de 100% quer dizer que deve-se marcar 8 clientes para uma caixa que atende 8 clientes. A fórmula simples para saber qual o número de clientes se deve marcar para cada caixa é dividir o número máximo que uma caixa pode atender pelo fator de penetração. Exemplo: { 8 / 60% } :. 8 / 0,6 = 13,333..., para o exemplo deve-se marcar 13 casas para uma caixa em uma área com fator de penetração de 60%.
